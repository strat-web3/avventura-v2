# Avventura v2

Text-based RPG game.

You can read the API docs [here](https://github.com/strat-web3/avventura-v2/blob/main/API_DOCS.md).

## Install

```bash
pnpm i
```

## Run

Create a `.env` file:

```
cp .env.template .env
```

Add your own keys in the `.env` file (you can get it in your [Reown dashboard](https://cloud.reown.com/)), then:

```bash
pnpm dev
```
